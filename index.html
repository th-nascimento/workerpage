Página de Cadastro de Funcionários

Esta página foi desenvolvida para uma empresa fictícia que precisa gerenciar diferentes tipos de funcionários, como gerentes e desenvolvedores. Siga as instruções abaixo para utilizar a página:

Como Utilizar

  1. Preencha o formulário com as informações do funcionário.
  2. Clique no botão "Cadastrar" para registrar o funcionário.
  3. Interaja com os cards dos funcionários para executar ações específicas.

Tecnologias Utilizadas

  • HTML
  • CSS (Bootstrap)
  • JavaScript

Validação dos Dados do Formulário e Tratamento de Exceções

Os dados inseridos no formulário serão validados pela função dataValidation(). Em caso de qualquer erro durante a execução do código, as exceções serão tratadas utilizando a função registerEmployee(). Esta função exibirá uma mensagem de erro tanto na página quanto no console.
